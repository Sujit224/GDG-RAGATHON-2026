# Statement 1: Fine Print Decoder 🧊

This directory contains the modernized "Fine Print Decoder" solution.

## 🏗️ Full-Stack Architecture & Tech Stack
To maximize code quality and user experience, we have upgraded this application into a decoupled client and server architecture.
- **Frontend:** React + Vite SPA. Features a premium glassmorphic UI, custom Chatbox, typing animations, and modern aesthetics using Vanilla CSS.
- **Backend API:** FastAPI exposing RESTful connections (`POST /api/chat`).
- **Data Ingestion:** PyMuPDF (`fitz`) for granular PDF parsing. We implemented a custom section and clause tracker during text extraction.
- **RAG & Orchestration:** Built with Langchain.
- **Vector Database:** ChromaDB to locally persist embeddings for fast similarity search.
- **LLM Engine:** OpenAI `gpt-4o-mini` with `text-embedding-3-small`.

## ✨ Features & Bonus Claims
1. **ELI5 Explanations:** The system leverages Langchain prompts explicitly designed to decode "legalese" into simple terminology.
2. **[BONUS CLAIMED] Source Attribution:** Our ingestion script (`src/ingest.py`) parses the document structure to extract **Page Numbers** and **Section/Clause Headers**. These metadata headers are stored in the vector database alongside the chunk texts. The LLM response is forced via prompt engineering to cite the *exact* Section and Page number for every single claim, ensuring maximum transparency.
3. **Advanced Aesthetics:** Modern design elements like dynamic iOS-style message bubbles, dark-mode gradient typography, and state-driven components.

## 🚀 How to Run

1. Navigate to the repository root `GDG-RAGATHON-2026` and install backend dependencies:
```bash
pip install -r requirements.txt
```

2. Create a `.env` file in the root directory (use `.env.example` as a template) and add your OpenAI API Key:
```env
OPENAI_API_KEY=your_key_here
```

3. Run the data ingestion script to format the PDF and build your Chroma database:
```bash
python Statement-1-Insurance-Decoder/src/ingest.py
```

4. **Start the Backend API:** In the first terminal window, boot the FastAPI server using Uvicorn:
```bash
cd Statement-1-Insurance-Decoder
uvicorn src.api:app --reload --port 8000
```

5. **Start the Frontend UI:** Open a second terminal window, install npm packages, and launch React:
```bash
cd Statement-1-Insurance-Decoder/frontend
npm i
npm run dev
```

Visit the `http://localhost:5173/` address generated by Vite to chat with the Bot!
